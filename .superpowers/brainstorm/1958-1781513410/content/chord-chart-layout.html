和弦谱展示视图 — 布局方案

核心功能：歌词与上方和弦一对一居中对齐，和弦标签悬浮于对应音节之上

方案 A — 等宽歌词，和弦居中

适合等宽字体或每字等宽的中文场景。和弦标签严格居中对齐每个字/音节。

方案 A - 对齐视图
[C]             [F]       
一闪一闪亮晶晶
[F]              [C]  
满天都是小星星
[C]        [G]       
挂在天空放光明

方案 B — 流式文本，和弦内嵌

和弦标签直接嵌入歌词文本上方，使用 flex 或 grid 对齐。适合非等宽字体。更接近传统吉他谱/简谱的呈现方式。

方案 B - 流式视图
C F
一闪一闪亮晶晶
F C
满天都是小星星
C G
挂在天空放光明

方案 C — 和弦谱 + 时间轴侧栏

在方案 B 基础上，左侧或上方增加滚动同步的时间轴，显示和弦进行与音频播放位置的对齐。适合边听边看的场景。

方案 C - 带时间轴
0:00 0:08 0:15 0:22

一闪一闪亮晶晶

满天都是小星星
